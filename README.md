## üåü Project Overview
EcoGenius is an innovative **Multi-Agent System (MAS)** designed as a powerful decision-support tool for modern sustainable supply chains.

It addresses the crucial challenge of balancing **Operational Costs** against **Environmental Impact (Carbon Emissions)**. Using intelligent agents, the system simulates complex logistics scenarios to pinpoint the most sustainable *and* profitable strategy.

### üéØ Key Impact Areas
* **Decarbonization:** Provides data-driven pathways to significantly reduce supply chain carbon footprint.
* **Cost Efficiency:** Identifies hidden inefficiencies and optimal routes/strategies for cost savings.
* **Strategic Planning:** Supports executive decision-making with interactive, clear visualization of trade-offs.

## üöÄ Get Started: Installation Guide

To run the EcoGenius simulator locally, follow these steps:

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/Gayatrip1702/EcoGenius-Simulator.git](https://github.com/Gayatrip1702/EcoGenius-Simulator.git)
    cd EcoGenius-Simulator
    ```
2.  **Activate Virtual Environment:**
    ```bash
    python -m venv venv
    .\venv\Scripts\activate
    ```
3.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
4.  **Run Application:**
    ```bash
    python app.py
    ```
5.  **Access:** Open your browser and navigate to `http://127.0.0.1:5000`
